# MUNTechinterface (Sapphire MUN)

This project is a rebuilt Model United Nations (MUN) management system, focusing on robust backend architecture, realtime synchronization, and administrative efficiency.

## ⚠️ Important Note
**Everything has been rebuilt from the ground up.**
The core architecture, database schema, and backend logic are now fully implemented and stable. 

**Note on UI:** The user interface (UI) is still to be built out and polished. The current focus has been on ensuring the technical foundation is flawless.

## Project Overview
- **Core Technology:** Next.js 14+, Supabase, TypeScript.
- **Realtime:** Implemented using Supabase Realtime for instant committee updates.
- **Admin Tools:** Custom administrative dashboard for delegate and session management.

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up environment variables: `.env.local`
4. Run development server: `npm run dev`
